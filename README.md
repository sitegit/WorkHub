# Приложение WorkHub
Приложение для просмотра списка вакансий, является тестовым заданием.

## Использованные Технологии
- Kotlin
- XML
- Clean Architecture
- MVVM
- Coroutines + Flow
- Retrofit 2
- Dagger 2
- Room
- Jetpack Navigation
- Multi-Module
